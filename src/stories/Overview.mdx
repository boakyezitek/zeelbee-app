### Typography
Font Family: We uses Poppins and Montserrat.

#### Poppins:
This is use in the app as the main Typography to handle p, span, b, a tags. This set in tailwind as 'zeel-default-font'.

#### Montserrat:
This is use in the app as the main Typography to handle heading tags. h1, h2, h3, h4 and h5. This set in tailwind as 'zeel-regular-font'.
